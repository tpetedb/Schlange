# Schlange

**Was?** Python. But in German. Because Smutzige Hansie sprecht kein anglisch.

Schlange is a Germanized Python preprocessor. You write code using German keywords, Schlange translates them to real Python at the token level, and everything runs for real. No tricks. No eval hacks. Just pure, unnecessary German engineering.

`verkuendet("Hallo Welt)` becomes `print("Hallo Welt)`. You're welcome.

## What can Schlange do?

### 1. Write Python in German (the obvious one)

```python
# hello.schl.py
defn begruessung(name, laut=Falschlich):
    sofern laut:
        gibzurueck f"HALLO {name.upper()}!!!"
    gibzurueck f"Hallo {name}"

verkuendet(begruessung("Welt"))
verkuendet(begruessung("Smutzige Hansie", laut=Wahrlich))
```

```bash
$ schlange run examples/hello.schl.py
Hallo Welt
HALLO SMUTZIGE HANSIE!!!
```

35 keywords. 25 built-in functions. All German. All unnecessary. All beautiful.

### 2. Parse 6 years of Spotify history

Because what's a farewell without data?

Schlange parses your Spotify Extended Streaming History (all those JSON files you forgot you downloaded) and ranks your top 2000 tracks into **Tom 2000** -- the definitive playlist of your listening career.

```bash
schlange run tools/tom2000.schl.py
```

Output:

- `tom2000_rangliste.csv` -- 2000 tracks, ranked by a composite score nobody asked for
- `tom2000_playlist_seed.json` -- track + artist + URI pairs
- `tom2000_uris.json` -- Spotify URIs for playlist creation
- `tom2000_zusammenfassung.txt` -- stats like "you listened to 5,700 hours of music, are you okay?"

### 3. Generate an AI farewell video

9 scenes of pure chaos generated by Google Veo 3.1. Explosions. German Python keywords floating in space. A cat coding. An Oktoberfest finale. All stitched together with Barbieboy as the soundtrack.

```bash
schlange run tools/make_video.schl.py
```

- Reads scene prompts from `assets/assets.json`
- Generates clips via Veo 3.1 API ($0.40/clip, total cost: ~$3.60)
- Stitches everything with ffmpeg
- Adds background music because silence is for quitters

The final video ends with a massive explosion. Because of course it does.

### 4. Send a tri-lingual farewell email

A farewell party email in German, English, and Dutch -- with the video attached. Because one language is never enough.

```bash
# Dry-run (preview, don't send)
schlange run tools/go.schl.py -- --skip-video

# Actually send it
schlange run tools/go.schl.py -- --skip-video --send
```

- HTML email with inline images and hazard-stripe headers
- Video attached (compressed to fit Gmail's 25 MB limit)
- Recipient list via `.env` or CSV file
- Dry-run by default because accidents happen

### 5. Run the entire pipeline with one command

The master orchestrator. One button to rule them all:

```bash
schlange run tools/go.schl.py -- --send
```

What it does:

1. Parses Spotify data (Tom 2000)
2. Generates video script + subtitles
3. Extracts poster frames
4. Generates 9 AI video clips (Veo 3.1)
5. Stitches + overlays background music
6. Generates tri-lingual email (DE/EN/NL)
7. Validates 40 mandatory content checks
8. Sends the email with video attachment
9. You close your laptop and walk away

## Quick Start

```bash
git clone https://github.com/tpetedb/Schlange.git
cd Schlange
python -m venv .venv && source .venv/bin/activate
pip install -e .
cp .env.example .env  # Fill in your API keys

# Run something
schlange run examples/hello.schl.py

# See the Python output
schlange transpile examples/hello.schl.py
```

- A Google AI API key (for Veo 3.1 -- your wallet was warned)

## How it works

```
.schl.py file --> tokenize --> match NAME tokens --> rewrite to Python --> exec
```

Schlange uses Python's `tokenize` module to find exact keyword matches and replace them. Strings, comments, and partial matches are never touched. You can mix German keywords with regular Python freely. All standard libraries and third-party packages work exactly as expected.

It's overengineered. It's unnecessary. It works perfectly.
